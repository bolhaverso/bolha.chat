Matrix

Você pode acessar nosso Matrix de qualquer cliente compatível com a rede através do endereço bolha.chat.

Essa é nossa rede principal de Chat.

Conheça mais sobre o projeto em https://matrix.org/

Servidor Matrix: bolha.chat

Não sabe oq é matrix? Leia nosso post!

Hydrogen

Você pode acessar nosso Matrix de um cliente minimalista que vai fazer você se sentir em um aplicativo de IM.

Element

O element é um cliente para acessar a rede Matrix. Você pode instalar no seu desktop, smartphone ou usar o cliente web.

A bolha tem um cliente web disponível no endereço https://element.bolha.chat.

Saiba mais sobre o projeto em https://element.io
